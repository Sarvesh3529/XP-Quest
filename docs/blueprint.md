# **App Name**: XP Quest

## Core Features:

- Gamified Task Management: Create, complete, and delete tasks, earning XP based on difficulty. Persists to local storage.
- Daily Task Locking: Lock tasks for the day, committing to them and enabling progress tracking.
- Workout and Streak Tracking: Log daily workouts to maintain a workout streak, which resets upon a missed day or logged rest day. The persistence of these stats requires careful consideration in code
- Boss Quests: Special weekly and monthly quests with high XP bonuses for larger achievements.
- Rank Progression: Earn XP to increase rank and unlock new rank badges. Displays all the ranks the user has achieved.
- Motivational Messages: Generate motivational messages based on wake-up time using a Genkit tool.
- Monthly Progress Reports: Generate monthly progress reports using AI based on completed tasks using a Genkit tool.

## Style Guidelines:

- Primary color: Neon Green (#39FF14) to reflect energy, progression, and the gamified nature of the application.
- Background color: Dark Gray (#1E1E1E) to provide a contrast to the neon and evoke a sense of a digital world.
- Accent color: Electric Blue (#7DF9FF) for secondary CTAs and additional highlights.
- Body and headline font: 'Space Grotesk', sans-serif. The request indicates that Space Grotesk is already used. 'Space Grotesk' provides a modern, tech-inspired feel to align with the overall theme.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use icons from Lucide-react that visually align with the quest categories and reflect the difficulty levels.
- Implement subtle animations for completing tasks, ranking up, and workout streak milestones.
